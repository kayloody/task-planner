# task-planner

## Description

A simple task planner app built with Vue.js. Integrates with
[task-planner-api](https://github.com/kayloody/task-planner-api)

## Installation

```bash
$ npm install
```

## Environment Variables

```bash
# task-planner-api URL
VUE_APP_API_URL
```

## Running the app

```bash
# Development
$ npm run serve

# Production
$ npm run build
```
